E-House

2019 | Java
Smart House Window System

The project is designed for changing window’s status automatically according to the weather by using openweathermap API, JSF(JavaServer Faces), Database System and AWS(Amazon Web Services). By using the openweathermap API, I obtain the current weather. Then I create an application of House Window System by applying the knowledge of computing design in architecture and programming language in Computer Science to calculate the weather status and automatically and remotely change the ventilation, color and opening range status of the house window
